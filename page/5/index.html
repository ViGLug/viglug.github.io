Vulnerabilità "Heartbleed"!

10/04/2014

E’ stato trovato un importante bug di sicurezza in un software ampiamente utilizzato, chiamato OpenSSL che si stima sia utilizzato da due terzi dei siti web mondiali, oltre che dai server di posta elettronica e da molti altri servizi, come quelli di messaggistica, chat e reti private virtuali. E’ fondamentale per gli utenti cambiare le password dei siti web affetti dal problema, tra cui: Google, Yahoo, Facebook, Flickr, Wikimedia, Dropbox, Tumblr, Lastpass, SoundCloud ed Amazon Web Service.
